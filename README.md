Keucher Mod is a DELTARUNE speedrun practice mod

# Installation Guide

1. Download the zip file from the releases: https://github.com/nhaar/keucher-mod/releases/latest
2. Unzip the file
3. Windows only: Run the .bat file for the version you want to patch (Linux or PC versions)
4. Not windows: You will need to apply the .bps manually. This online patcher can be used: https://www.marcrobledo.com/RomPatcher.js/, you will need to load the appropriate .bps file, which is located in the patch_files folder inside the zip, and select the appropriate data.win. Keep it in mind that for the full-game versions, the individual chapter data.wins need to be named `data_keucher.win`
5. When you want to use the mod, you will need to change the main data.win of the game. You will want to rename the vanilla data.win to something else, and then rename the keucher mod data.win to simply data.win

[This video showcases roughly the manual installation process](https://www.youtube.com/watch?v=KisS3tkutsM)

# Usage
Welcome to Keucher Mod!

This mod contains many features to speedrun the game. You will need some time learn all the features useful to you, but as a starter, you should know that pressing the mouse's Right Button will open the game menu. There, you can look at all the features, learn what they do, enable the ones you want or not. You can also look at the value for all the keybinds, and reassign them as you wish.
